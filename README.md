# AA002-兴趣班要不趁早

对于有孩子的家庭来说，教育是永恒的话题。近年来国外的各种教育理念不断涌入中国，前几年针对国内孩子学习太辛苦的现象涌现出一群快乐教育的观点，强调要尊重孩子的天性，不要过早扼杀孩子对学习的兴趣。这几年，精英教育的观点又占据各大头条，批判快乐教育是穷人的选择，而国外中产以上的孩子比国内的孩子压力更大，学习更努力。

### 2.3 github中各板块的使用指南

- 小标题：changelog：小写不空格

- 内容：人名+动词+名词：表明谁做了什么
